# 📊 Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Sales Dashboard developed using Power BI to analyze sales performance across different regions, products, and customer segments. It helps users identify trends, monitor key performance indicators (KPIs), and make data-driven business decisions.

## 📝 Project Description
The dashboard provides a clear visualization of sales data through interactive charts, filters, and KPIs. It enables users to track revenue, profit, quantity sold, and regional performance while uncovering valuable business insights for improved decision-making.

## 🛠️ Tech Stack
- Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

## 📂 Data Source
- Sample Sales Dataset (Excel/CSV)
- The dataset contains information on sales, customers, products, orders, regions, profit, and revenue used for dashboard analysis.

## ✨ Features
- 📈 Interactive sales performance dashboard
- 🌍 Region-wise and category-wise sales analysis
- 💰 Revenue, Profit, and Sales KPI tracking
- 🔍 Dynamic filtering using slicers
- 📊 Easy-to-understand visualizations for business insights
